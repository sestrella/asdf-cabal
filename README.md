# asdf-cabal

Cabal plugin for asdf version manager.

## Requirements

- Python 3

## Usage

Install the plugin:

```
asdf plugin-add cabal https://github.com/sestrella/asdf-cabal.git
```

List all available versions:

```
asdf list-all cabal
```

Install a specific version:

```
asdf install cabal <version>
```
